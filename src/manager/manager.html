Umfrage-Manager — Hilfe

Was ist das? Der Umfrage-Manager ist eine lokal laufende Anwendung zum Erstellen, Verteilen und Auswerten von Umfragen — ganz ohne eigenen Server. Sie legen hier Umfragen an, verteilen sie an Befragte und werten die zurückerhaltenen Antworten hier aus.

Zwei Wege, eine Umfrage zu verteilen:

  • HTML-Client ("Als HTML exportieren"): erzeugt eine eigenständige HTML-Datei pro Befragten, die vollständig offline funktioniert. Geeignet, um die Datei direkt (z. B. per E-Mail) an einzelne Personen zu verteilen.
  • Gehosteter YAML-Client ("Als YAML exportieren"): Sie geben nur die YAML-Datei weiter; alle Befragten öffnen stattdessen dieselbe, feste Client-Adresse (unten) im Browser und importieren dort die Datei. Praktisch, wenn mehrere Befragte dieselbe Datei nacheinander ausfüllen und weiterreichen — dann benötigt nicht jede Person eine eigene HTML-Datei.
