Umfrage-Client — Hilfe

Was ist das? Dieses Dokument ist ein digitaler Fragebogen. Sie können ihn direkt in Ihrem Browser ausfüllen.

Pflichtfelder sind mit * markiert und müssen ausgefüllt sein, bevor Sie exportieren können.

Wie funktioniert es?

  • Füllen Sie das Pflichtfeld (Hochschule) aus.
  • Beantworten Sie alle Fragen.
  • Klicken Sie auf Antworten exportieren (YAML) — eine YAML-Datei wird heruntergeladen.
  • Senden Sie diese YAML-Datei zurück an die Person, die Ihnen den Fragebogen geschickt hat.

Ihre Antworten werden automatisch in Ihrem Browser gespeichert (localStorage), solange Sie den Fragebogen nicht schließen. Sie können eine zuvor exportierte (auch teilweise ausgefüllte) YAML-Datei wieder importieren, um an ihr weiterzuarbeiten — das Versionsfeld wird dabei automatisch um eins erhöht, ist aber nur ein Indiz und kein Echtheitsnachweis.

Umfrage-Client

Dieser zentrale Umfrage-Client hat aktuell noch keine Umfrage geladen. Importieren Sie eine .yaml-Datei, die Sie von der Umfrage-Ersteller:in oder einer/einem Kolleg:in erhalten haben.

Nach dem Import erscheint der Fragebogen und Sie können ihn ausfüllen. Ihre Antworten werden anschließend in diesem Browser gespeichert; die YAML-Datei können Sie danach wieder exportieren und weitergeben.
